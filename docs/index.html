    프로젝트 1 : 위키사이트 상단바 색상 고정 브라우저 확장프로그램

    해당 설정 위키 사이트의 상단바를 사용자의 개인 설정으로 바꿀 수 있음.
    그런데 해당 위키 사이트의 사용자가 개별 문서에서 css를 사용할 수 있음.
    이 css 스타일은 사용자 개인 설정보다 우선함.
    그런데 css의 사용이 사이트 상단바에도 적용.
    이것을 해결하기 위해 '위키사이트 상단바 색상 고정 브라우저 확장프로그램'을 제작.

    개인 설정을 들고오는 법은 몰라서 일단 따로 설정하는 팝업을 만듦.
    그런데 이 이상은 모르겠어서 보류.
