    프로젝트 1 : 위키사이트 상단바 색상 고정 브라우저 확장프로그램

    해당 설정 위키 사이트의 상단바를 사용자의 개인 설정으로 바꿀 수 있음.
    그런데 해당 위키 사이트의 사용자가 개별 문서에서 css를 사용할 수 있음.
    이 css 스타일은 사용자 개인 설정보다 우선함.
    그런데 css의 사용이 사이트 상단바에도 적용.
    이것을 해결하기 위해 '위키사이트 상단바 색상 고정 브라우저 확장프로그램'을 제작.

    개인 설정을 들고오는 법은 몰라서 일단 따로 설정하는 팝업을 만듦.
    그런데 이 이상은 모르겠어서 보류.

    프로젝트 2 : 트위터(현 X) 사이드바 제거

    현재 트위터 사이드바 ui는 쓸모없는 것까지 덕지덕지 있는 더러운 상태
    이것을 해결하기 위해 제작 구상.

    프로젝트 3 : 메모 프로그램

    어디까지나 구상.
    메모를 함에 있어 그림이 필요할 때가 다수.
    그러나 일반 메모앱/프로그램을 쓰자니 그림을 사용 못하고
    그림 프로그램을 사용하자니 텍스트 작성에 있어서 직접 쓰거나 텍스트 도구를 이용해야함.
    텍스트 도구가 매우 느리고 불편해서 결국 직접 써야 하는데,
    쓸게 많으면 손이 아파 그림과 텍스트 메모를 모두 편하게 할 수 있는 메모 프로그램을 구상.
